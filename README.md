## Estrutura do Repositório

```
site-institucional/
├── index.html       → Pilar 1 — Estrutura-Mãe (HTML semântico, acessível, auditável)
├── style.css        → Pilar 2 — Estética de Rigor (tipografia, cinematografia, blindagem visual)
├── script.js        → Pilar 3 — Motor Cinético (scroll, reveal, linhas animadas)
├── .htaccess        → Protocolo de Acesso A+ (HSTS, CSP, HTTPS, segurança de infraestrutura)
└── README.md        → Documentação institucional deste repositório
```

## Princípios Técnicos

- **Segurança:** Certificação A+ (SSL Labs) via `.htaccess` com HSTS 2 anos + CSP blindado
- **Performance:** Zero dependências pesadas — HTML/CSS/JS puros + Google Fonts
- **Acessibilidade:** WCAG 2.1 AA | `lang="pt-pt"` | semântica HTML5 completa
- **Privacidade:** Sem cookies de rastreio | Sem analytics externos | Anti-coleta por design
- **Licença:** [EUPL 1.2](https://joinup.ec.europa.eu/collection/eupl/eupl-text-eupl-12)

## Deploy

Deploy directo em servidor Apache via `ptservidor.pt`.  
O ficheiro `.htaccess` é activado automaticamente pelo Apache — sem configuração adicional.
